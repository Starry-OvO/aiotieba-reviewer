## 简介

基于[**aiotieba**](https://github.com/Starry-OvO/aiotieba)实现的百度贴吧高弹性吧务审查框架

## 安装

```shell
pip install aiotieba-reviewer
```

## 教程

[**云审查教程**](https://starry-ovo.github.io/aiotieba-reviewer/tutorial/reviewer/)
